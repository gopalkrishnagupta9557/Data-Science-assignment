# 🏏 International T20 Cricket Data Analysis

## 📌 Project Overview

This project performs **data analysis on International T20 cricket match data** using Python data science libraries.
The objective of this project is to explore the dataset, clean the data, perform statistical analysis, and create meaningful visualizations.

The analysis is implemented in a notebook using **Python in Google Colab**, making it easy to reproduce and understand.

---

## 🎯 Objectives

* Perform **data cleaning and preprocessing**
* Apply **Pandas and NumPy functions**
* Conduct **statistical analysis**
* Create **data visualizations**
* Understand patterns in international T20 matches

---

## 🗂 Dataset

The dataset used in this project:

**International_T20_Data.csv**

It contains information such as:

* Match venues
* Cities
* Match outcomes
* Winning teams
* Run margins
* Wicket margins
* Toss decisions
* Match dates

## 📊 Data Analysis Steps

### 1️⃣ Data Loading

The dataset is loaded using Pandas.

### 2️⃣ Data Exploration

Basic dataset exploration includes:

* Checking shape of dataset
* Viewing columns
* Understanding data types
* Generating statistical summaries

### 3️⃣ Data Cleaning

* Handling missing values
* Removing duplicates
* Renaming columns
* Creating new features

### 4️⃣ Statistical Analysis

NumPy functions used:

* Mean
* Median
* Standard deviation
* Variance
* Maximum and minimum values

### 5️⃣ Data Aggregation

Using Pandas operations:

* `groupby`
* `value_counts`
* `pivot_table`
* Sorting and filtering data

### 6️⃣ Data Visualization

Visualizations created include:

* Bar charts
* Histograms
* Pie charts
* Count plots
* Box plots
* Heatmaps

These visualizations help in identifying trends in match results and team performance.
